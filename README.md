# LandingPage

## Description

Page de présentation d’une application fictive nommée TaskFlow. Cette landing page statique présente un hero, des fonctionnalités, des témoignages, des offres tarifaires et un appel à l’action.

## Fichiers

- `index.html` : page principale de la landing page.
- `styles.css` : styles de la mise en page.
- `images/` : dossier pour l’image de présentation.

## Fonctionnalités

- Section hero avec bouton d’appel à l’action.
- Cartes de fonctionnalités.
- Témoignages clients.
- Offres tarifaires.
- Section CTA finale.

## Utilisation

1. Ouvrir `index.html` dans un navigateur.
2. Vérifier que `styles.css` est présent.
3. Vérifier que l’image `images/dashboard.png` est disponible.

## Notes

Page statique sans logique JavaScript côté client.
